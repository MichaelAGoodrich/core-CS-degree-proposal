# Impact on Existing Courses (Under Construction)

**Note:** This document is under construction.

## Rationale

This section documents anticipated impacts on existing courses and staffing as the revised degree structure is implemented.

### Key Impacts

- **Junior core scheduling:** The junior core classes are intended to be taught only in **fall and winter** semesters, which concentrates demand into two terms and requires coordinated scheduling.
- **Teaching allocation:** The three new required courses for the junior core will require **additional teaching allocation**, impacting staffing plans and course rotations.
- **Elective enrollments:** The **breadth + depth** elective structure may shift student demand across current 400-level courses, affecting enrollment patterns and potentially changing which electives run each term.
