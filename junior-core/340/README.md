# CS 340: Software Design

This folder documents CS 340 as it appears in the junior core. The course is included as-is because it is already required and meeting its intended outcomes.

## Supporting Character and Professional Development

Beyond technical competency, CS 340 contributes to student **character building** by engaging with ethical and professional dimensions of software design:

- **Responsibility to users**: Design decisions affect user experience, security, and accessibility. Students learn that technical choices have human consequences
- **Teamwork and communication**: Design work requires articulating ideas to colleagues, receiving critique, and iterating toward shared solutions
- **Ethical decision-making**: Students encounter real tensions: Should we simplify for speed or engineer for maintainability? How do we balance features with security? These decisions develop professional judgment
- **Long-term thinking**: Understanding design patterns, architecture principles, and technical debt helps students think beyond the immediate assignment to the lifetime of a system

These dimensions support students' growth in professional integrity and responsibility, core components of character development.
