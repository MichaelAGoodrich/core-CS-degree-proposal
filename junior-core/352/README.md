# CS 352: Database Systems

This folder documents CS 352 (renumbered from CS 452) as it appears in the junior core. The course advances the proposal's emphasis on CS as the **science of abstraction**, with data abstractions as a foundational layer for modern computing systems.

## Role in the Abstraction Spine

CS 352 highlights how data abstractions enable scalable, reliable, and meaningful computation. Students learn how data models, query languages, and system tradeoffs represent real-world information in precise and actionable forms, reinforcing the program's focus on enduring abstractions rather than transient tools.

## Current Course Overview

The current course covers core database and data-systems concepts, including:

- Relational modeling and SQL
- Schema design, normalization, and constraints
- Indexing, storage, and query processing
- Transactions, recovery, and reliability
- Distributed data systems and consistency tradeoffs
- Large-scale data processing and pipelines

Databases and data abstractions are also emphasized in the proposed ACM curriculum, reinforcing the importance of this course within the core.
