# CS 330: Programming Languages

CS 330 is a **proposed addition** to the junior core. While the course currently exists, it is not required today; the committee recommends adding it to the core because reasoning about programming languages is a foundational abstraction in computer science.

**Course Description:**
Principles and concepts characterizing high-level computer programming languages, process and data abstraction, encapsulation, inheritance, functional programming, logic programming, scanners, and parsers.

## Rationale for Inclusion

The ability to reason about programming languages helps graduates select the right language and concepts for a given problem, rather than being constrained to a single paradigm (for example, object-oriented reasoning). This supports both **theory and practice**: students gain conceptual models for computation and language design while also learning how language features shape real-world software design, correctness, and maintainability. This aligns with the unit review emphasis on strength in both theoretical scope and practical experience.
